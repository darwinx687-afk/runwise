# Runwise Documentation

Runwise is an open-source readiness, tracing, replay and eval toolkit for AI Agents, MCP servers and LLM applications.

This documentation is plain Markdown for the open-source preview. There is no docs build system in this loop.

## Current Boundaries

Runwise works from source today. It does not provide a hosted service, database, login, cloud sync, model runtime, agent framework, npm release, or marketplace listing yet.
