# Ecosystem Compatibility

Runwise detects local signals for common AI project ecosystems:

- MCP
- LangChain
- OpenAI Agents
- Dify
- browser-use
- Claude Code
- Codex
- Cursor
- Windsurf
- Ollama
- OpenAI-compatible APIs
- China-ready LLM providers

Detection is local and heuristic. Runwise looks at file names, package metadata, config names, environment examples, and lightweight docs.

Runwise does not execute these frameworks, import runtime integrations, call models, call external APIs, or claim official partnerships.
