## Dev.to Intro

Runwise is a local-first toolkit for making AI agent readiness visible before teams ship demos, CI gates, or early releases.

The first public preview, `v0.1.0-preview.0`, focuses on a small but reviewable workflow: run Doctor checks, validate local traces, generate static replay reports, and convert reviewed failures into eval case drafts.

This release is source-install only. It does not publish npm packages, provide hosted SaaS, sync to cloud storage, add telemetry, or claim official ecosystem integrations.

Release: https://github.com/darwinx687-afk/runwise/releases/tag/v0.1.0-preview.0
