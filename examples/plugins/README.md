# Plugin Examples

These are documentation-only examples for a possible future Runwise plugin architecture.

Runwise does not load these plugin files yet. They are not executable in current Runwise.

The examples contain:

- draft `runwise.plugin.json` manifests
- draft `rules.json` rule packs
- no secrets
- no API keys
- no network calls
- no dependencies
