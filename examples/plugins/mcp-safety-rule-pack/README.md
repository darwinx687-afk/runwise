# MCP Safety Rule Pack

This is a documentation-only plugin example.

Runwise does not load this plugin yet. It is not executable in current Runwise.

## What It Shows

This draft rule pack shows how a future local JSON rule pack could describe:

- shell-like tool detection
- filesystem write tool detection
- delete/send/payment naming risk
- approval recommendations

## Files

- `runwise.plugin.json`: draft plugin manifest
- `rules.json`: draft JSON rule pack

## Safety Notes

- No secrets.
- No real API keys.
- No network calls.
- No dependencies.
- No executable plugin code.
