# 插件示例

这些是未来 Runwise 插件架构的文档示例。

Runwise 目前还不会加载这些插件文件。它们在当前 Runwise 中不可执行。

这些示例包含：

- draft `runwise.plugin.json` manifest
- draft `rules.json` 规则包
- 不包含 secrets
- 不包含 API key
- 不会发起网络调用
- 不需要依赖

- [国内大模型 provider 规则包](./china-ready-provider-rule-pack/README.zh-CN.md)
