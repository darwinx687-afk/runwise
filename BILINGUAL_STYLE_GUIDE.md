# Bilingual Style Guide

Runwise uses English as the primary project language and Simplified Chinese as a mirrored documentation language.

## English

- Prefer direct, technical language.
- Use "AI agent", "MCP server", and "LLM application" consistently.
- Keep phase boundaries explicit.

## Simplified Chinese

- Prefer clear product engineering language.
- Translate "readiness" as "就绪度".
- Translate "tracing" as "追踪".
- Translate "replay" as "回放".
- Translate "eval" as "评测".

## Mirroring Rule

When a root README section changes, update the matching section in the other language.
